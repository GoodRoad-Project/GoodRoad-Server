**Про докер-контейнер**
Для того, чтобы запускать докер-контейнер из консоли WSL необходимо:
* Либо непосредственно установленный докер на WSL
* Либо если он на винде, то выполниеть следующую интсрукцию settings -> recourses -> WSL integration -> enable integration with my default WSL distro + выбрать свою убунту -> apply&restart


**Про pom.xml**
Если зависимости не подтянулись автоматически, то есть pom.xml весь горит красным и в других файлах названия пакетов тоже красные, то вот решение:
```
rm -rf ~/.m2/repository/org/springframework/boot
mvn clean install
```

Сначала стоит попробовать просто вторую команду. Если не получилось, то сначала первую, потом вторую сделать.  Вторая команда работает довольно долго, должна завершаться без ошибок. Сразу после этого файл по-прежнему может гореть красным - надо выйти и снова зайти в IDEA
