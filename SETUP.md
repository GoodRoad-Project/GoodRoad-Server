**Про докер-контейнер**
Для того, чтобы запускать докер-контейнер из консоли WSL необходимо:
* Либо непосредственно установленный докер на WSL
* Либо если он на винде, то выполниеть следующую интсрукцию settings -> recourses -> WSL integration -> enable integration with my default WSL distro + выбрать свою убунту -> apply&restart

**Как запустить контейнер с сервером и бд?**
```
docker compose up --build
```
Ничего очищать после каждого запуска по дефолту не требуется. Что для первого запуска, что для последующих - команда одинаковая. Она не пересобирает проект с нуля, только новый код собирает

**Как зайти в бд в контейнере?**
```
docker exec -it goodroad_db psql -U {DB_USER} -d {DB_NAME}
```
